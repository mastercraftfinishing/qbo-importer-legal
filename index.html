Privacy Policy

Effective Date: March 18, 2026
Application: QBO Sales Order Importer
Version: 1.0
Summary (Plain English) This application runs entirely in your browser. We do not collect, store, or share any of your personal data or business data. Your QuickBooks credentials and order data never leave your browser except to be sent directly to Intuit's official QuickBooks Online API.

1. Overview

This Privacy Policy describes how the QBO Sales Order Importer ("Application", "we", "us") handles information when you use our Application. We are committed to protecting your privacy and handling your data with transparency.

This Application is a private, internal-use tool designed to import sales order data from Excel or CSV files into QuickBooks Online. It is not a public-facing commercial product.

2. Information We Do NOT Collect

The Application does not collect, store, transmit, or share any of the following:

  • Personal identification information (name, email, address, phone number)
  • QuickBooks API credentials, Client IDs, Client Secrets, or Access Tokens
  • Business data, order data, customer names, or financial information
  • Usage analytics, behavioral data, or telemetry
  • Browser data, cookies, or device identifiers
  • IP addresses or location data

3. How Data is Processed

All data processing occurs entirely within your local web browser. Specifically:

  • File Processing: Excel and CSV files you upload are parsed entirely in your browser's memory using client-side JavaScript. No file data is sent to any server operated by this Application.
  • API Credentials: Your QuickBooks API credentials (Client ID, Client Secret, Access Token, Realm ID) are stored only in your browser's active session memory and are cleared when you close the browser tab or window.
  • QuickBooks Data: Sales order data is sent directly from your browser to Intuit's official QuickBooks Online API servers using your own credentials. This Application does not act as an intermediary server.

4. Third-Party Services

The Application communicates exclusively with the following Intuit-operated API endpoints:

  • Sandbox: https://sandbox-quickbooks.api.intuit.com
  • Production: https://quickbooks.api.intuit.com

Data transmitted to these endpoints is governed by Intuit's Privacy Policy, available at intuit.com/privacy/statement. We encourage you to review Intuit's privacy practices.

The Application also loads the SheetJS (XLSX) library from Cloudflare's CDN (cdnjs.cloudflare.com) to process spreadsheet files. This is a read-only library load; no user data is sent to Cloudflare.

5. Data Security

Because the Application does not transmit or store your data on any external server, the primary security responsibility rests with you. We recommend:

  • Using the Application on a secure, trusted device and network
  • Not sharing your QuickBooks API credentials with unauthorized parties
  • Closing the browser tab when you have finished using the Application to clear session data
  • Regularly rotating your Intuit API Access Tokens per Intuit's security guidelines

6. Children's Privacy

This Application is intended for business use only and is not directed at children under the age of 13. We do not knowingly collect any personal information from children.

7. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated effective date. We encourage you to review this policy periodically.

8. Your Rights

Because we do not collect or store any personal data, there is no personal data for us to provide, correct, or delete. If you have questions about data held by Intuit (QuickBooks), please contact Intuit directly through their privacy portal.

If you have questions or concerns about this Privacy Policy or the Application's data practices, please contact the Application owner.
